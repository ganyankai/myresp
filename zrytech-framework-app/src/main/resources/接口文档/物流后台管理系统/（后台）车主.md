## 3.管理员 - 车主审批 ##

**备注**


**URL**
>/admin/carCargoOwner/approveCarOwner

**请求参数**

    {
    	"params": {
            "businessId": "",             (必填)（int）车主Id
    		"content": "",                (必填)（str）审批意见
     		"result": ""                  (必填)（str）商品结果（字典取值）
    	},
    	"token": ""
	}

**测试入参**

    {
    	"params": {
            "businessId": "1",
    		"content": "审批通过",
     		"result": "agree"              
    	},
    	"token": "admin"
	}

**响应示例**

    {
        "code": 1,
        "msg": "success",
        "data": "审批成功"
    }

## 5.管理员 - 修改车主免审核状态 ##

**备注**


**URL**
>/admin/carCargoOwner/updateCarOwnerAvoidAudit

**请求参数**

    {
    	"params": {
            "id": "",             (必填)（int）车主Id
            "avoidAudit": ""      (必填)（boolean）审核状态
    	},
    	"token": ""
	}

**测试入参**

    {
    	"params": {
            "id": "1",
            "avoidAudit": "false"
    	},
    	"token": "admin"
	}

**响应示例**

    {
        "code": 1,
        "msg": "success",
        "data": "修改成功"
    }
