## 4.修改密码 ##

**URL**

>/customer/api/updatePassword

**请求参数**
     
    {
        "params": {
                        "oldPassword": "A123456.",          （必填）（str），原密码
                        "newPassword": "A123456#",          （必填）（str），新密码
                        "confirmPassword": "A123456#"       （必填）（str），确认密码
         },
        "token": "" （必填）（str）
    }
**测试入参**

    {
    	"params": {
                "oldPassword": "A123456.",
                "newPassword": "A123456#",
                "confirmPassword": "A123456#"
         },
    	"token": ""
    }


**响应示例**    
    {
        "code": 1,
        "msg": "success",
        "data": null
    }
