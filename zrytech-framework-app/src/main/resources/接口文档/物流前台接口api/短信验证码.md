## 短信验证码 ##

## 1.腾讯短信 ##

**URL**
>/tx/verifcode

**请求参数**
    
    {
    	"params":{
    		"phone":"17612345678"  （必填）（str）手机号
    	}
    }

**响应示例**

    {
	    "code": 1,
	    "msg": "success",
	    "data": null
    }
