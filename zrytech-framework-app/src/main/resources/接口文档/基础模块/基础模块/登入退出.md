
**本文档接口提交方式统一为：POST**

### 1.用户登录


**URL**
>/login

**请求参数**
```json
{
  "params": 
{
	"userCount":"zhangsan",  //用户账号
    "password":"123456"      //用户密码
  },
}
```

**响应示例**
``` json
{
    "code": 1,
    "msg": "success",
    "data": "token_1525850724013"  
}
```

###2.用户退出

**URL**
>/logout

**请求参数**
``` json
{
   "token": "string"
}
```  

**列表信息响应示例**
``` json
   {
      "code": 1,
      "msg": "success",
      "data": null
   }
```  


### 2.用户修改密码

**URL**
>/user/savepassword

**请求参数**
``` json
{
	"params": 
    {
		"userCount":"zhangsan",       //用户账号
	    "password":"123456"           //用户旧密码
        "newPassword":"12345678"      //用户新密码
	 },
   "token": "string"
}
```  

**列表信息响应示例**
``` json
   {
      "code": 1,
      "msg": "success",
      "data": null
   }
```
