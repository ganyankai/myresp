###6,文章点赞与收藏
**url:**
>/articlelikecollect/save

**请求参数**
``` json
{
	"params":{
             "articleId":5
             "type":1         //0 点赞  1收藏
             },
    "openid": "string",
    "token": "string"
}
``` 
**响应：**
``` json
{
    "code": 1,
    "msg": "success",
    "data": null
}
```
